To launch a project,<i> you need to </i>import it into the terminal. To do this, specify the https address that will appear when you click the "code" button above the project files in github. We specify the address in the terminal after the git clone command . We write all this to the folder where we want to import the code.

After importing the files, you need to open the root folder in the terminal and install all the dependencies using the "npm install" command.
